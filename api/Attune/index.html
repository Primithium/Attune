Attune

a tiny utility module for pitching sounds with 12-TET.

Properties​

Effect​

This item is read only and cannot be modified. Read Only
</>
Attune.Effect: PitchShiftSoundEffect

a reference to the PitchShiftSoundEffect linked to this Attune object. for almost all use cases this should not be modified, however, its properties can be.

Tuning​

</>
Attune.Tuning: number

the offset (in cents) to apply to the sound before any operation. intended to be used for fine tuning samples.
